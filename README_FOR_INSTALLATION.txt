To install the Legume version provided for the CPC submission (1.0.1), enter in the legume folder. There you should be able to locate the setup.py file. From this directory:

pip install -e ./

To run inverse design you have to install autograd with the command:

pip install autograd

To run all examples you need to install jupyter notebook with the command:

pip install notebook 

then use jupyter notebook and navigate to the docs/examples folder.

For full documentation with API reference, go to directory  legume/docs/_build/html and open the index.html file.
